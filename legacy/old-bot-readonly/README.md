## Installing Environment on Windows

Install [Node JS](https://nodejs.org/) <br>
Install [NVM](https://github.com/coreybutler/nvm-windows/releases) <br>

## Installing Environment on Linux

Install [Node JS](https://nodejs.org/de/download/package-manager) <br>
Install [NVM](https://github.com/nvm-sh/nvm) <br>

## Installing Node Modules

Run in Terminal: ```npm install``` or ```npm i```

## Running it

For Testing: ```npm test```<br>
For Running: ```npm start``` <br>
For Running in dev mode: ```npm start dev``` <br>
To Build a Docker Container: ```npm build``` <br>
